# v2.0.0

## 版本亮点

- 新增华奋达（Weikav）双 8K 方案机械键盘电量读取与绑定能力。
- 新增中英文界面切换，常用设置与状态文案支持双语显示。
- 更新应用图标与界面演示图，整体 GUI 展示更完整。

## 新增功能

- 支持扫描键盘候选设备，并在设置界面中完成手动绑定与解绑。
- 绑定成功后，键盘电量信息可进入设置界面与托盘状态显示链路。
- 设置项支持持久化保存，重启后仍可保留当前偏好与绑定信息。

## 体验改进

- 更新应用图标与演示资源，下载页和 README 展示与当前 GUI 更一致。
- 设置界面支持中英文切换，便于不同语言环境下使用。

## 升级说明

- 如果你正在使用华奋达（Weikav）双 8K 机械键盘，请在设置界面中重新执行一次键盘绑定流程。
- 若系统托盘未立即显示最新状态，可在托盘菜单中手动执行一次刷新。
